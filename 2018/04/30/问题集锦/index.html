Kali 2018.2 默认安装的node 不含npm

apt-get 安装不了 npm , 这是因为新版 nodejs 中 npm 默认包含，也就没有单独安装 npm 的需要了。但是 kali 默认安装的node 中不含 npm。于是尝试找以前的包，手动安装。试了一下，没有安装成功。于是使用了另一种方法。

在 nodejs 官网上下载 node 安装包，解压后找到目录下的 lib/node_modules/npm/bin/npm-cli.js 直接使用。

为了方便使用可以在~/.bashrc 添加一个别名 npm，这样就能使用 npm 进行安装包了。
